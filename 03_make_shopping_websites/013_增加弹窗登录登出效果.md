### Step 2:

打开app/helpers/application_helper.rb,将代码修改为如下内容

```ruby app/helpers/application_helper.rb
module ApplicationHelper
  def resource_name
    :user
  end

  def resource
      @resource ||= User.new
  end

  def resource_class
      User
  end

  def devise_mapping
      @devise_mapping ||= Devise.mappings[:user]
  end  
end
```
