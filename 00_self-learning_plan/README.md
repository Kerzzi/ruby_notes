##  路线三：HTML/CSS/JavaScript

* 3.1 [《Head First HTML与CSS(第2版)》](https://book.douban.com/subject/25752357/)(必学)（Done）
  * 非常入门的一本书,对新手非常友好。 虽然很厚，700多页。但是图表很多，3-4天足以看完，HTML与CSS的入门好书。不过如果不愿意买书，也可以跟着[这个网站：freecodecamp](https://www.freecodecamp.cn/map) 上面的内容去做。这个网站内容免费，但是极为优秀！极为优秀！极为优秀！
* 3.2 《CSS揭秘》(必学)
  * 3.2.1 极其惊艳的一本书，涵盖了CSS3的很多奇技淫巧，虽说有些地方不太实用，但让人眼前一亮，很值得看。
* 3.3 《JavaScript DOM编程艺术》(必学)

* 3.4 《javascript语言精粹》(必学)
  * 3.4.1 俗称蝴蝶书。超薄的一本，半天就可以看完。js是一门很混乱的语言，这本书能够帮助你区分哪些是语言的精华，哪些是糟粕，对于语言精华，应该深入学习。糟粕部分能看懂别人写的代码就行，自己就不用尝试了。
  * 3.4.2 JavaScript语言精粹_读书笔记
* 3.5 《你不知道的JS》(必学)
  * 3.5.1 精彩至极的一本书，将JavaScript的坑一网打尽。之前搞不懂的问题，比如闭包，this之类的都可以在这里找到答案
* 3.6 《JavaScript高级程序设计（第3版）》(选修)
  * 3.6.1 无需看完，可以当做工具书,俗称红宝书。前七章是重中之重，必须反复阅读，直至完全理解。DOM，事件流，表单，JSON，Ajax与最后几章也相当重要。其余章节可以略读。
* 3.7 完成了codecademy上的JavaScript教程，并做课后配套的几个网站项目
* 3.8 《JavaScript设计模式与开发实践》(选修)
  * 3.8.1 设计模式是必须了解的内容，这本是写得不错的
* 3.9 AJAX 教程(必学)
  * 3.9.1 Ajax全接触
* 3.10 学习JQuery，跟着http://try.jquery.com走一遍，学到的知识在项目里使用(必学)
* 3.11 《锋利的jQuery (豆瓣)》(必学)
  * 3.11.1 这本书只需要看 1-9 章就够了，也就是 290 页左右的内容
  * 3.11.2 jQuery基础(四)—动画篇（选修）
  * 3.11.3 jQuery 教程（选修）
  * 3.11.4 jQuery 实战经典（选修）
* 3.12 阮一峰老师的《ES6 标准入门》(必学)
* 3.13 选学内容：
  * 3.13.1 慕课网：HTML+CSS基础课程（选修）
  * 3.13.2 《重新介绍 JavaScript（JS 教程）》（选修）
